# React Todo List Demo

## Overview
A React-based todo list application with API integration.

## Features
- React components with hooks
- State management
- API integration with Flask backend
- CRUD operations

## Setup

```bash
# Install dependencies
npm install

# Start development server
npm start
```

## Project Structure
```
15-react-todo/
├── src/
│   ├── App.js
│   ├── components/
│   │   └── TodoList.js
│   └── index.js
└── package.json
```

## Usage
1. Start the Flask backend API (from part2/12-flask-basic-api.py)
2. Start React app: `npm start`
3. Open http://localhost:3000

## Note
This is a simplified demo. For full implementation, see the frontend folder in the main demo-projects directory.
