- **Two Pointers**: Two Sum, Valid Palindrome

- **Tree Traversal**: Inorder, Preorder, Postorder
- **Binary Search Trees**: Search, Insert, Delete
- **Graph Algorithms**: BFS, DFS, Shortest Path

- **1D DP**: Fibonacci, Climbing Stairs, House Robber
- **2D DP**: Unique Paths, Edit Distance, Longest Common Subsequence
- **Advanced DP**: Knapsack, Coin Change, Word Break

#### **Linked Lists (15% of problems)**
- **Basic Operations**: Reverse, Merge, Cycle Detection
- **Advanced**: Remove Nth Node, Intersection, Copy with Random Pointer

#### **Sorting & Searching (10% of problems)**
- **Binary Search**: Classic search, Search in Rotated Array
- **Sorting Algorithms**: Quick Sort, Merge Sort, Heap Sort
