## ï¿½ Python Symbols & Syntax Names Reference

**Complete Guide to Python Symbols - Know What They're Called!**

### **Brackets & Parentheses**

| Symbol | Name | Python Usage | Example |
|--------|------|--------------|---------|
| `()` | **Parentheses / Round Brackets** | Function calls, Tuples, Grouping | `print()`, `(1, 2, 3)` |
| `[]` | **Square Brackets / Brackets** | Lists, Indexing | `[1, 2, 3]`, `list[0]` |
| `{}` | **Curly Braces / Braces** | Dictionaries, Sets | `{"key": "value"}`, `{1, 2, 3}` |
| `<>` | **Angle Brackets** | Type hints, Comparison | `List[int]`, `x < y` |

### **Operators & Arithmetic**

| Symbol | Name | Meaning | Example |
|--------|------|---------|---------|
| `+` | **Plus / Addition** | Add numbers, Concatenate strings | `5 + 3`, `"Hi" + " there"` |
| `-` | **Minus / Subtraction / Hyphen** | Subtract, Negative number | `10 - 5`, `-10` |
| `*` | **Asterisk / Star / Multiply** | Multiply, Repeat, *args | `5 * 3`, `"Hi" * 3`, `*args` |
| `/` | **Forward Slash / Division** | Divide (float result) | `10 / 3` â†’ `3.333` |
| `//` | **Double Slash / Floor Division** | Divide (integer result) | `10 // 3` â†’ `3` |
| `%` | **Percent / Modulo / Remainder** | Get remainder | `10 % 3` â†’ `1` |
| `**` | **Double Asterisk / Exponent / Power** | Power, **kwargs | `2 ** 3` â†’ `8`, `**kwargs` |
| `=` | **Equal / Assignment** | Assign value | `x = 10` |
| `==` | **Double Equal / Equality** | Compare equal | `x == 10` |
| `!=` | **Not Equal / Inequality** | Compare not equal | `x != 10` |

### **Comparison Operators**

| Symbol | Name | Meaning | Example |
|--------|------|---------|---------|
| `<` | **Less Than** | Is less than | `5 < 10` â†’ `True` |
| `>` | **Greater Than** | Is greater than | `10 > 5` â†’ `True` |
| `<=` | **Less Than or Equal** | Is less or equal | `5 <= 5` â†’ `True` |
| `>=` | **Greater Than or Equal** | Is greater or equal | `10 >= 10` â†’ `True` |

### **Punctuation & Special Characters**

| Symbol | Name | Python Usage | Example |
|--------|------|--------------|---------|
| `.` | **Dot / Period / Decimal Point** | Decimal numbers, Method calls, Module import | `3.14`, `list.append()`, `import math` |
| `,` | **Comma** | Separate items | `[1, 2, 3]`, `print(a, b)` |
| `:` | **Colon** | Start block, Dict key-value, Slicing | `if x:`, `{"key": "value"}`, `[1:5]` |
| `;` | **Semicolon** | Separate statements (rarely used) | `x = 1; y = 2` |
| `'` | **Single Quote / Apostrophe** | String | `'Hello'` |
| `"` | **Double Quote / Quotation Mark** | String | `"Hello"` |
| `'''` or `"""` | **Triple Quotes** | Multi-line string, Docstring | `"""Long text"""` |
| `` ` `` | **Backtick** | Not used in Python | (Used in other languages) |

### **Special Symbols**

| Symbol | Name | Python Usage | Example |
|--------|------|--------------|---------|
| `#` | **Hash / Pound / Number Sign** | Comment | `# This is a comment` |
| `\` | **Backslash** | Escape character, Line continuation | `\n` (newline), `\t` (tab) |
| `_` | **Underscore** | Variable names, Ignore value, Private | `my_var`, `_, x = divmod(10, 3)` |
| `__` | **Double Underscore / Dunder** | Magic methods | `__init__`, `__str__` |
| `@` | **At Sign** | Decorator | `@property`, `@staticmethod` |
| `|` | **Pipe / Vertical Bar** | Bitwise OR, Set union | `x | y`, `set1 | set2` |
| `&` | **Ampersand** | Bitwise AND, Set intersection | `x & y`, `set1 & set2` |
| `^` | **Caret** | Bitwise XOR, Set symmetric diff | `x ^ y`, `set1 ^ set2` |
| `~` | **Tilde** | Bitwise NOT | `~x` |
| `!` | **Exclamation / Bang** | Not (only in !=) | `x != y` |
| `?` | **Question Mark** | Not used in Python | (Used in ternary in other languages) |

### **Logical & Boolean**

| Symbol/Word | Name | Meaning | Example |

| `in` | **IN** | Check membership | `"a" in "apple"` |
| `is` | **IS** | Identity check | `x is None` |

### 2ï¸âƒ£ String Indexing & Slicing (15 minutes)

**Indexing - Access Single Character:**

```python

print(text[1])   # Y - second character
print(text[5])   # N - last character

# Backward indexing (starts at -1)

print(text[-2])  # O - second last
print(text[-6])  # P - first character

# Out of range causes error
# print(text[10])  # Error! Index out of range
```

**Slicing - Extract Multiple Characters:**

**Syntax:** `string[start:end:step]`
- start: where to begin (included)
- end: where to stop (NOT included)
- step: how many to skip (optional)

```python
text = "PYTHON"

# Basic slicing
print(text[0:3])   # PYT (index 0,1,2)
print(text[2:5])   # THO (index 2,3,4)

# Shortcuts
print(text[:3])    # PYT (from start to 3)
print(text[2:])    # THON (from 2 to end)
print(text[:])     # PYTHON (complete string)

# Negative indices
print(text[-4:])   # THON (last 4 characters)
print(text[:-2])   # PYTH (all except last 2)

# Step (skip characters)
print(text[::2])   # PTO (every 2nd character)
print(text[::3])   # PH (every 3rd character)
print(text[::-1])  # NOHTYP (reverse!)

# Practical examples
email = "user@gmail.com"
username = email[:4]        # user
domain = email[5:]          # gmail.com
print(f"Username: {username}")
print(f"Domain: {domain}")
```

```
[2:5]  = THO (2,3,4)
```

```python
    my_list.append(item)
    return my_list
```

| **Tuple** | âœ… Yes | âŒ No | âœ… Yes | âœ… Yes | Fixed data, faster than lists |

### âš ï¸ Common Mistakes to Avoid
```python

    content = file.read()
```

```python
# âŒ WRONG - Catching wrong exception

    num = int("abc")

# âœ… CORRECT
try:
    num = int("abc")
```

**Example 3: Simple Recommendation System**
```python
import pandas as pd

# Movie ratings by users
ratings = pd.DataFrame({
    'user': ['Alice', 'Alice', 'Bob', 'Bob', 'Carol'],
    'movie': ['Action1', 'Comedy1', 'Action1', 'Horror1', 'Comedy1'],
    'rating': [5, 3, 5, 4, 4]
})

# Find similar users
def recommend(user_name):
    # Get user's preferences
    user_ratings = ratings[ratings['user'] == user_name]
    
    # Find users who liked same movies
    similar_users = ratings[
        ratings['movie'].isin(user_ratings['movie'])
    ]
    
    # Get their other movies
    recommendations = similar_users[
        ~similar_users['movie'].isin(user_ratings['movie'])
    ]['movie'].unique()
    
    return recommendations

# Usage
print(f"Recommendations for Alice: {recommend('Alice')}")
```
