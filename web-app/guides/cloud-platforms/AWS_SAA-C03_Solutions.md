## AWS Services Quick Reference

| Service | When to Use |
|---------|-------------|
| **S3** | Object storage, static websites, data lakes, backups |
| **S3 Glacier / Deep Archive** | Long-term archival, infrequent access, compliance |
| **S3 Intelligent-Tiering** | Unpredictable access patterns, auto cost optimization |
| **EC2** | Virtual servers, full OS/networking control |
| **Lambda** | Serverless functions, event-driven, <15 min tasks |
| **RDS** | Managed relational DB (MySQL, PostgreSQL, Oracle) |
| **Aurora** | High-perf MySQL/PostgreSQL, up to 5x faster |
| **Aurora Serverless** | Variable workloads, auto-scaling, pay-per-use DB |
| **DynamoDB** | NoSQL key-value, millisecond latency |
| **ElastiCache** | In-memory caching (Redis/Memcached) |
| **CloudFront** | CDN, global content delivery at edge locations |
| **Route 53** | DNS, routing policies, health checks, failover |
| **SQS** | Message queuing, decoupling, buffering |
| **SNS** | Pub/sub messaging, fan-out notifications |
| **ECS/EKS** | Container orchestration (Docker/Kubernetes) |
| **Fargate** | Serverless containers, no infra management |
| **VPC Endpoint** | Private access to AWS services (no internet) |
| **NAT Gateway** | Internet for private subnets (outbound only) |
| **Transit Gateway** | Connect multiple VPCs and on-prem |
| **Direct Connect** | Dedicated private connection to AWS |
| **WAF** | Web firewall (SQL injection, XSS, rate limiting) |
| **Shield** | DDoS protection (Standard=free, Advanced=paid) |
| **Secrets Manager** | Store and auto-rotate credentials |
| **KMS** | Encryption key management |
| **CloudTrail** | API call logging and auditing |
| **AWS Config** | Resource config tracking and compliance |
| **DataSync** | Fast online data transfer to AWS |
| **Storage Gateway** | Hybrid storage (on-prem cache + S3) |
| **Snowball/Snowcone** | Offline bulk migration (10TB+) |
| **DMS** | Database migration with minimal downtime |
| **Kinesis** | Real-time streaming data |
| **Athena** | Serverless SQL on S3 data |
| **Glue** | ETL jobs and data catalog |
| **Step Functions** | Workflow orchestration, state machines |
| **RDS Proxy** | Connection pooling (great with Lambda) |
| **Global Accelerator** | Improve global perf via AWS backbone |
| **AWS Backup** | Centralized backup management |
| **EventBridge** | Event routing, scheduled tasks |
