# Weather Dashboard Demo

## Overview
A React-based weather dashboard with API integration.

## Features
- Weather API integration
- React components
- Charts and visualizations
- Responsive design

## Setup

```bash
# Install dependencies
npm install

# Get API key from OpenWeatherMap
# Add to .env file: REACT_APP_WEATHER_API_KEY=your_key

# Start development server
npm start
```

## API Setup
1. Sign up at https://openweathermap.org/api
2. Get your API key
3. Add to environment variables

## Usage
1. Start React app: `npm start`
2. Open http://localhost:3000
3. Enter city name to get weather

## Note
This is a simplified demo. For full implementation, see the frontend folder in the main demo-projects directory.
